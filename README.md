This repository represents the source code for the Mechanical Core Squadron Designer. This is a tool created for use with the Mechanical Core Combat tabletop strategy game system. All rights to MCC are held and decided by its core developer. This tool is developed with permission and is intended to be made available for the use of and further development by the game's creator and community.

Feature plan
* Workspace Selection
  * New workspace creation includes a pop-up
  * Once created a workspace exists as a SQLite database on disk with a unique name
* Mecha editor (Mecha Workshop)
  * Implements data structure based on spreadsheet's data structure
  * Stretch goal: attach mecha icon/image as per worksheet
* Pilot Editor (Pilot Dossiers)
* Team creator (Squadron Editor)
  * Pull from saved mecha
  * Stretch goal: go to the Make New Mecha window and be able to come back to the team creator
* Squadron overview pane (Squadron Roster)
  * Presents the squadron and the mecha in it in various helpful, easy to read formats
  * Stretch goal: export to excel in something modeled after the template
  * Stretch goal: include mecha icons/images
